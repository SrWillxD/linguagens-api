## ⚙️ Funcionalidades
- GET de linguagens
- POST de linguagens
- PUT de linguagens
- DELETE de linguagens

## 🛠️ Tecnologias utilizadas
- Java 18
- Spring Framework
- MongoDb

## 🖥️ Endpoints
#### GET Linguagens e POST Linguagens
````
http://localhost:8080/linguagens
````
#### DELETE Linguagem e PUT Linguagem
````
http://localhost:8080/linguagens/{id}
````
#### GET lingugens pelo id
````
http://localhost:8080/linguagens/{id}
````
